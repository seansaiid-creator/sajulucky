[사주행운 수정 파일 안내]

이번 파일은 기존 기능과 디자인을 변경하지 않고, 아래 항목만 보강한 덮어쓰기용 파일입니다.

1. robots.txt
- 개인정보처리방침 / 이용약관 차단 제거
- sitemap 경로 유지

2. sitemap.xml
- 기존 기본 페이지 + SEO 페이지 + seo-content.html 포함

3. sitemap-seo-pages.xml
- SEO 10개 페이지에 lastmod / changefreq / priority 추가

4. about.html
- 기존 최소 페이지를 현재 사이트 디자인 톤에 맞춰 보강

5. contact.html
- 기존 최소 페이지를 현재 사이트 디자인 톤에 맞춰 보강
- 문의 이메일 반영: seansaiid@gmail.com

6. seo-content.html
- 기존 최소 페이지를 긴 정보형 콘텐츠 페이지로 보강
- AdSense 및 검색엔진 관점의 콘텐츠 보완용
